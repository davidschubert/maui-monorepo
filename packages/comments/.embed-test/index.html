Ein fremder Blog-Artikel

Diese Seite simuliert eine Drittseite (anderer Origin, anderer Stack — hier pures HTML). Unten lädt der Maui-Comments-Loader das Widget als iframe. Aufruf z. B.: python3 -m http.server 4999 in diesem Ordner, dann http://localhost:4999/?widget=http://localhost:3001.

Der Query-Param ?widget=<origin> bestimmt die Widget-Instanz (Default http://localhost:3001); &theme=dark und &primary=rose werden ans Script durchgereicht.
